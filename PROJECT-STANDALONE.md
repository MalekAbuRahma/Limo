# VIP limousine CARS — Monthly Tracker

This is a **separate copy** of the tracking system. The original Honda accord project is unchanged.

## Run

```bash
npm install
```

Double-click **`START-VIP-limousine-CARS.bat`**  
App: http://localhost:3000/

## Stop

**`STOP-VIP-limousine-CARS.bat`**

## Data

- Database: `data/taxi.db` (own copy — not shared with other projects)
- Vehicle name default: **VIP limousine CARS**

## Save another backup

**`SAVE-SNAPSHOT.bat`** creates `vip-limousine-cars-saved-YYYY-MM-DD_HHMM` under `carrent/`
